This project performs an end-to-end Exploratory Data Analysis (EDA) on a structured Data Warehouse using SQL. Working across the Gold Layer (fact_sales, dim_customers, dim_products), the analysis covers:

• 🗓️ Date & Range Exploration — sales timeline, customer age distribution
• 📐 Measures Overview — total revenue, quantity sold, average price, order count
• 🌍 Magnitude Analysis — customers by country & gender, revenue by category, items sold per region
• 🏆 Ranking Analysis — top/bottom 5 products by revenue, top 10 highest-value customers, least active customers

Built to translate raw warehouse data into actionable business insights using pure SQL.
